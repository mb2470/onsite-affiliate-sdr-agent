# Supabase Backend for AI SDR Agent
   
   This directory contains the database schema and migration files for the Supabase backend.
   
   ## Files
   - `schema.sql` - Complete database schema
   - `migrations/` - Migration files (if needed)
   
   ## Setup
   1. Create Supabase project
   2. Run schema.sql in SQL Editor
   3. Import contact database CSV
   
   See `/docs/MIGRATION_GUIDE.md` for full instructions.
